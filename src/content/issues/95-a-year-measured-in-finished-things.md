**Status:** Complete after the calendar year.

Classify real outputs:

| Thing | Finished unit | Date | Collaborators | Still requires maintenance |
|---|---|---|---|---|
| | | | | |

Separate:

- Completed projects
- Completed phases
- Repairs returned to service
- Public work delivered
- Experiments concluded
- Work abandoned deliberately
- Work produced by collaborators or AI systems

Do not count file creation as completion when the intended public or physical
result did not occur. Do not erase maintenance because it resembles repetition.
